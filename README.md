# Dotpulse.Visibility Package for Neos CMS #

## Composer

### Integration Repository
```
composer config repositories.visibility vcs ssh://hg@bitbucket.org/dotpulse/neos-plugin-visibility
```

### Installation
```
composer require dotpulse/visibility
```
